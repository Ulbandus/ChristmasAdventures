## How to play?

Linux:

```sh
apt install git python3
git clone https://github.com/Ulbandus/ChristmasAdventures/
cd ChristmasAdventures
pip install -r requirements.txt
python3 ChristmasAdventures 
```

## Controls

* Left: Move left  
* Right: Move right  
* Up: Jump   
* Down: Sit
